Responsive Design

Qu'est-ce que le Responsive Design ?

Le Responsive Design (ou design adaptatif) est une approche de conception web qui permet à un site de s'adapter automatiquement à la taille de l'écran.

Les breakpoints

Un breakpoint est un seuil de largeur d'écran à partir duquel on modifie l'apparence de la page web afin de l'adapté au support (mobile, tablette, dekstop).

Liste des breakspoints

  • Mobile => max-width : 480px (- ou égal à 480)
  • Tablette => max-width : 768px
  • Laptop (ordi portable) => max-width : 1024px
  • Desktop (Les tours) => min-width : 1025px

Attention !!!

La résolution des écrans en css (nb de px en largeur-hauteur) est représenté en pixel logique.

Dans la réalité, les appareils sont en pixel physique.

Par exemple: Un Iphone 13 a une résolution logique de 390 x 844 pixels mais en réalité, elle a une résolution physique de 2532 x 1170px.

Par exemple pour un écran Rétina, 3px physique pour 1px logique.

Principe fondamentaux du Responsive Design

Utilisation des % ou vh/vw au lieu de valeurs fixes comme le pixel.

Utiliser des propriétés tel que max-width: 100% pour que les images ne dépassent jamais de leur conteneur.

Les media queries => Ce sont des instructions qui appliquent des styles en fonction de la taille de l'écran.

Actuellement nous vivons dans un monde où les téléphones portables sont homis présents. Il est conseillé MAINTENANT de concevoir sont site en MOBILE FIRST

Qu'est ce que le mobile first ?

C'est un concept de web design optimisé pour le mobile, il consiste à concevoir un site en mettant la priorité sur la version mobile et ensuite d'adapter le site progressivement sur des écrans plus larges.
